# Shelf Builder

3D wardrobe configurator built with Next.js, Three.js, and React Three Fiber. Design custom wardrobes with shelves, dividers, and panels, then export specifications to PDF.

## Setup

Copy `.env.example` to `.env` and configure:

```bash
cp .env.example .env
```

Required environment variables are defined in `.env.example`.

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
